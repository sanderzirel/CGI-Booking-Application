# CGI Booking Application

This is project for CGI summer internship.

Author: Sander Zirel

## Project setup:
  - Spring Boot (Java 21)
  - H2 Database
  - Vue 3

## How to run:
- Open terminal and go to the backend folder using  **cd .\backend\\**
- Run backend with command   **.\mvnw spring-boot:run**
- Open antoher terminal and go to the frontend folder using  **cd .\frontend\\**
- Run frontend with command  **npm run dev**
- You can view project on http://localhost:5173/ now.
