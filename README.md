# CGI Booking Application

This is project for CGI summer internship.

Author: Sander Zirel

## Project setup:
  - Spring Boot (Java 25)
  - H2 Database
  - Vue 3

## How to run:
- Make sure to download Node.js and Java JDK 25
- Clone the project with **git clone https://github.com/sanderzirel/CGI-Booking-Application.git**
- Open project in your IDE
- Open terminal and go to the backend folder using  **cd .\backend\\**
- Run backend with command   **.\mvnw spring-boot:run**
- Open another terminal and go to the frontend folder using  **cd .\frontend\\**
- Run command **npm install**
- Run frontend with command  **npm run dev**
- You can view project on http://localhost:5173/ now.
