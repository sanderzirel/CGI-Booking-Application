# Process

I decided to start with the frontend in order to build the foundation of the project. Once the basic structure of the frontend was ready, I began developing the backend so that I can fetch tables from the database and display them in the restaurant layout.

After that, the development became somewhat hectic because I didn't have a strict plan. I started switching between working on backend logic and frontend functionality. However, once most of the backend was implemented, development became easier since I had to focus on making things work on the frontend side.

# Difficulties

Developing TableController.java took a lot of time because it was challenging to design the logic for generating reservations.

The suggestions logic also required effort and time. AI tools helped me get started by giving hints and examples.

The TableReservationRepository was created with the help of AI because I did not know that JpaRepository makes database queries so much easier.

For the HomeView, creating the restaurant layout was a little tricky to figure out. Also, implementing suggestion constraints required work and testing to make sure that switching filters wouldn't produce unexpeced outcomes.

# The project

I estimate that I spent approximately 20–25 hours working on this project.

I used AI to help me guide through problems where I didn't know how to start. I also used websites such as **Stack Overflow** to solve specific problems, for example when implementing a dropdown menu.
